------------------------	SETUP		----------------------------------

step 1. Install Visual Studio with C++ development package.
step 2. Install C++ compiler and add it to environment variables
step 3. Create a new project in Visual Studio
step 4. change the debugger configuration to x64
step 5. Download openCV
step 6. Add yhe path of openCV to enviroment variables and in VS path
step 7. Restart computer

----------------------		USE		------------------------

1. Change the image name to ss.png
2. Save it in the same folder as of porject
3. Run the code
4. The code will prompt you to enter the angle at which you want to rotate
5. The rotated imgae will be displayed and original image will also be displayed too
